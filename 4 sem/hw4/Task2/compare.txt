сравнение времени сортировки отсортированного массива на обычной сортировке пузырьком и оптимизированной 
оптимизированная: n = 50   t = 2е-06
                  n = 100  t = 2е-06
                  n = 200  t = 5е-06



обычная : n = 50   t = 1.7e-05
          n = 100  t = 5e-05
          n = 200  t = 1.5e-04


 Видим что для неоптимизированной сортировки время больше и быстрее растет от количества элементов
